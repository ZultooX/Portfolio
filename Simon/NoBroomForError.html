No Broon For Error
This was the first project that we made during the time studying at The Game Assembly. Made in Unity Game Engine in a timespan of 6 weeks on half time, We used Race The Sun as our reference game to follow gameplay and make it similar.

Tree Editor

Making levels for a "infinite runner" and saving time can be hard. Making tools for the designers will decreases time on this process.

In short I designed a tool to place out trees that will fall in a sequence with many settings to tinker with, maybe too much for what it was used for.

Moon Timer

Displaying the time in a Traditional way, a clock counting down, is boring.
Race The Sun instead uses the sun in the background to show time left, Essentially
"You race the sun". If the sun fully setts, you die.


Our game is set at night so its now a moon instead. By simply lerping from a set start location to an end location with how long the run should take, the moon slowly sets behind the trees.

clothes Line

For decoration in the city stage a magicly moving clothes line looked cool in tests. Curving the clothes along the clothes line is made easy with a bezier curve with a reasonable speed.

Closing Thoughts

Being the first game made at the game assembly it turned out alright, the gameplay is fun but I think there are too many features packed into the game. One of the more unnessesary is the potion crafting system. It is way too har to look at what you want to craft while dodging all obstacles in your way.

The game is avaible on itch if you want to play it, No Broom For Error
