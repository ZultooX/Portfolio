Stella & Orbit: Take Flight
This was a challenging but fun project where i got to learn many new technologies. The time for this project was 15 weeks during half time. The reference game was Ratchet & Clank: Quest for Booty.

Cascaded Shadow Maps

With our games theme being space pirates we will have alot of open space in the level wich means long distance shadows. At the time we had a very simple shadow system that would only show shadows close to the player. So cascaded shadow maps was my mission to figure out. With help from Shadow Cascading LearnOpenGL It started off rough with my first shadow map looking like this
This issue took a while to solve being quite new too everything. The image looks like it has very much FOV which must mean that the view matrix is wrong. It was simple, two values in the view matrix were switched making the FOV value another value. What did help was DX11 Cascaded Shadow maps example

Drag to see the result and difference, left has no cascades and right has cascades

Text Renderer

Redering text is essenial for a custom game engine to display information for the player to look at or interact with. The chosen text library was freetype, a free and open source library. This was very hazzle free and was able to get a working prototype in just a few days.

I made a component from it and was now way easier to add text to the game. This was our final text Renderer that we used for the remaning projects.

Here is some text that I rendered in engine with comic sans font

NPC Dialouge

The game needed some more interactions in the world so some dialog NPCs were in order. Making it appear instantly looks weird so instead writing it out looks better.
Lastly to make it easily editable I created a component for the dialoge and export it with the scene export tool we used.
