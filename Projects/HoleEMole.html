Hole-É-Mole
First game that I worked on for mobile. Also the only mobile game that we made during The Game Assembly, only for android in this instance. This project had a time limit of 6 weeks on half time, same as Project 1. Our reference game was Lara Croft GO this project, a mobile and PC puzzle game.

Moving Platforms

Moving platforms are important for dynamic puzzle creating so these were an obvious feature we wanted. These will carry enities that are on the platform, Player, Enemies, Barrels, etc.

Creating them was quite a challenge because it was so early in development so all systems was not in place. The way it was done was by raycasting from the platform normals and moving whatever it hit.

As an after though we already register what is on the current tile so instead of raycasting being slow we could just move whatever we knew was on the platform.

Main Menu

Having a cool and good looking Main Menu is very important since this is the playes first impression of what your game will be. We wanted something that was like Lara Croft GO 3D Main Menu but ours, with a cave theme.

I made something im very proud off that also looks cool. It is a spining conveyor belt with a simple moving/rotation animation.

Level Select

The level select I think became great and using it felt great. Moving down physicaly into the mines on the map feels very good and makes the expecience more enjoyable.

For every number and X there is an invisible button that then leads to the selected Level and World.
The main reason to wanting to make it like this was a reference to Plant VS Zombies 1 and the achievment screen. There is an easter egg to where if you scroll beyond the achievments you can keep scrolling and find more refrences to different games. I loved this feature so I wanted somthing similar.

Closing Thoughts

It was very interesting to work on a mobile game since a mobile is not very powerful, you had to think of ways to optimise you might not have if you made the game for pc. The game also came out great and fun and easy to play in some offtime.
